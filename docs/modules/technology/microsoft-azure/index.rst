Microsoft Azure
===============

Microsoft Azure là nền tảng điện toán đám mây và cơ sở hạ tầng được thiết kế
bởi Microsoft. Azure cung cấp nhiều dịch vụ để quản lý, xây dựng và triển khai
các giải pháp phần mềm trên nền tảng internet.
